Class Aggregator<T1, T2>

Superclass for all data aggregators. Creates a structured method for pulling and normalizing data from the database into usable Aggregation objects.

remarks

Try not to not keep too instances of these classes alive, or you are likely to run out of memory. Aggregator.entity can be very large, especially if it contains many relations.

    database entity used to create an Aggregation
